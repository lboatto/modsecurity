ModSecurity

Web Application Firewall permettant de filtrer les requêtes HTTP avant qu'elles ne parviennent à l'application


Source : https://www.darknet.org.uk

Permet de bloquer les attaques les plus courantes :

  • SQL Injection (SQLi)
  • Cross Site Scripting (XSS)
  • Local File Inclusion (LFI)
  • Remote File Inclusion (RFI)
  • Remote Code Execution (RCE)
  • Module disponible pour Apache & NGINX
  • Gratuit
  • Open Source
  • https://www.modsecurity.org/

Intéressant...

  • Facile à installer
  • Ne nécessite pas de modifications de l'application
  • Mode apprentissage

Installation


                      $ apt install libapache2-modsecurity modsecurity-crs

                      $ mv /etc/modsecurity/modsecurity.conf-recommended \
                      /etc/modsecurity/modsecurity.conf

                      $ systemctl restart apache2
